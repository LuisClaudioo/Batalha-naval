# Batalha Naval em C

Projeto desenvolvido em linguagem C para praticar lógica de programação e manipulação de matrizes.

## Tecnologias utilizadas
- Linguagem C
- Compilação via GCC
- Execução em terminal

## Descrição do projeto
O projeto simula parte do jogo Batalha Naval utilizando um tabuleiro representado por uma matriz 5x5.
Os navios são posicionados de forma vertical e horizontal, com exibição de suas coordenadas no console.

## Conceitos aplicados
- Lógica de programação
- Matrizes (arrays bidimensionais)
- Laços de repetição (for)
- Variáveis e controle de fluxo
- Organização de código em C

## Exemplo de funcionalidade
- Criação de tabuleiro 5x5
- Posicionamento de navio vertical
- Impressão das coordenadas do navio no terminal

## Status
Projeto acadêmico para fins de aprendizado.
